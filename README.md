## Project Description
AI Book Sphere is an interactive digital bookstore website designed to help users discover books by genre and author.  

## Pages / Site Structure
- **Home:** `index.html`  
  Introduction, group members, navigation menu, and banner image.
- **Books Catalog (Table Page):** `pages/books.html`  
  Displays a table of books with title, author, genre, rating, and price.
- **Signup / Preferences (Form Page):** `pages/signup.html`  
  Collects user name, email, genre preference, and reading options (HTML form only).
- **About:** `pages/about.html`  
  Explains the purpose of AI Book Sphere and how recommendations work.
